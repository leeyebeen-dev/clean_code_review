# JUnit 프레임워크

- 공통적인 접두어를 모두 제거하자.
- 명확한 표현을 위해 조건문을 캡슐화하자.
(조건문을 메서드로 뽑아내 적절한 이름을 붙이자.)
- 이름을 명확하게 붙이자.
- 부정문은 긍정문보다 이해하기 약간 더 어렵다.
부정문을 긍정문으로 표현해 조건문을 반전하자.
- 함수에 적합한 이름을 주어 가독성을 높이자.
- 함수를 분리하자
- 함수를 일관적으로 사용하자.

# ComparisonCompactor 모듈
